1. to test the apishka, you need to open the postman, and make a post request under the address http://localhost:8080/api/v1/auth/login
2. the request will be made like this:
{
 "email": "admin@mail.ru ",
 "password": "admin"
}
3. You must get a token, and this token must be specified in the header where "key = Authorization, and where value = the received token",
this step must be specified in any other requests
